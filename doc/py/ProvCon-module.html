Controller(path)

source code 

Create and return a new "Controller" socket.

A "Controller" is a process which listens for remote management requests to the Provisioning back-end services (such as TFTPD)

Parameters:
  • path (string) - path of the unix socket
Returns: multiprocessing.connection.Listener
the new Listener object
