Class Field

source code

object --+
         |
        Field

Field objects represent column meta data and provide data-conversion services.

Field objects are created when a connection is made to the database. Each Field object represents one column in the field_info table. There should be no need for these objects to be created by the user elsewhere.

Data conversion SQL {<->} PYTHON is also handled by Field objects.
